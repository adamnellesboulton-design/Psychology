The math behind the model

One equation, run with different settings. Illustrative, not fitted to data.

State

gi ∈ (0,1): the gain on relevance in channel i — near 1 is flooding, near 0 is collapse, ½ is euthymia.   a ∈ (0,1): the bar, a slow threshold each bid must clear — engaged in every condition, so it moves throughout; only bipolar crosses its Hopf.

Dynamics   (overdot = d/dt; S(x) = 1/(1+e−x), S′(0) = ¼)

τg ġi = −gi + S ( β(gi−½) + I − κ(a−½) + c(ḡ−gi) + σξi )
τa ȧ = ḡ − a

ḡ is the mean over channels; ξi ∼ N(0,1) drawn fresh each step; τg = 1, τa = 40, dt = 0.05 (Euler).

Parameters

βself-reinforcement (loop gain)κhomeostat strength (sets the bar)
Ienvironmental stressorcintegrating gain (coupling)
kdiscount steepness (ADHD)λflexibility (autism)

The stressor I is a steady outside push; an optional per-tick jitter adds a random kick of size up to δ each step. A valence ν ∈ [−1,1] leans those kicks: the kick is upward with probability (1+ν)/2, so I(t) = I0 ± δ U(0,1) with the sign biased by ν (ν > 0 toward flooding, ν < 0 toward collapse). It drives the time-series and field views.

Resting states & bifurcations   (single channel; at rest a = g)

effective gain   G = β − κ,    fixed points:   g = S( G(g−½) )

G < 4 → one euthymic state at g = ½  ·  G > 4 → two arms + an unstable threshold = fold (schizophrenia). Since S′(0) = ¼, the split is exactly at G = 4.

euthymia stability:   tr J = (β/4 − 1)/τg − 1/τa

tr J < 0 → damped (it rests)  ·  tr J > 0 with G < 4 → orbited = Hopf (bipolar). Near the fold recovery slows monotonically; near the Hopf it slows while ringing.

Miscalibration profiles   (stable loop, β small; the fault is a fixed setting)

ADHD:   v(d) = 1 / (1 + k d)  — value of a reward at delay d; weight = the resting g it drives
autism:   mismatch = g − g*,   the realized gain tracks the optimum g* by a fraction λ (λ = 1 cancels it)
A deliberately simple toy for the theory, not a fit to data. Under the hood it is one equation run with different settings; for the dynamicists, the two malfunctions are a fold (schizophrenia) and a Hopf bifurcation (bipolar).
